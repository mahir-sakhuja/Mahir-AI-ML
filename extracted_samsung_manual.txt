How to use the various modes of the washing machine

Last Update date : Feb 16. 2023

Innovations like Ecobubble™ and QuickDrive™ washing machines can work hard to clean clothes in a fraction of the time and while using less energy. You'll also spend less time doing the laundry and more time enjoying the things that you love.

It is important to select a suitable laundry course for clothes in order to finish the laundry well. The following table below outlines the basic washing modes of Samsung washing machines.

Using these courses properly will help to make your laundry dry much quicker and easier.

To start the course, you could follow these steps:

1. Press the Power button to turn on the washing machine.



2. Turn the Cycle Selector to select a cycle.

2.	Turn the Cycle Selector to select a cycle

3. Change the cycle settings (Temp, Rinse, and Spin ) as necessary.



4. To add an option, press Options. Use the button again to choose a preferred item.



5. The machine provides the easy-to-access buttons for Bubble Soak, Delay End, and Easy Iron for your convenience. If you want to use one of these options, press the corresponding button.



6. Press Start/Pause



And to change the cycle during the operation follow the below steps:

1.Press Start/Pause to stop the operation.



2. Select a different cycle.



3. Press Start/Pause again to start the new cycle.



Note:

The actual design of the control panel may differ based on the model you have.

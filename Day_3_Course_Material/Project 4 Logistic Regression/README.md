# Day 3: Employee Retention Predictor

This interactive Streamlit application implements a Logistic Regression model to predict the probability of an employee leaving the company based on satisfaction levels, project count, monthly hours, and salary.

## How to Run:
1. Navigate to this directory:
   ```bash
   cd "Day 3 Course Material/Project 4 Logistic Regression"
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch Streamlit:
   ```bash
   streamlit run app.py
   ```

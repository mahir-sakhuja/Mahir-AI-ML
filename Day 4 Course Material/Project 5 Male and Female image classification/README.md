# Day 4: Male & Female Image Classification using Logistic Regression

This interactive Streamlit application trains a standard Machine Learning model (Logistic Regression) on flattened image vectors to classify images into Male or Female.

## How to Run:
1. Navigate to this directory:
   ```bash
   cd "Day 4 Course Material/Project 5 Male and Female image classification"
   ```
2. Generate synthetic/dummy dataset for training:
   ```bash
   python create_dummy_dataset.py
   ```
3. Train the model:
   ```bash
   python train_model.py
   ```
4. Install app dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Launch Streamlit:
   ```bash
   streamlit run app.py
   ```

# Day 5: Male & Female Image Classification using Convolutional Neural Networks (CNN)

This interactive Streamlit application trains a Deep Learning model (CNN) using Keras and TensorFlow to classify images into Male or Female.

## How to Run:
1. Navigate to this directory:
   ```bash
   cd "Day 5 Course Material/Project 7 Male and Female image classification using CNN"
   ```
2. Generate synthetic/dummy dataset for training:
   ```bash
   python create_dummy_dataset.py
   ```
3. Train the model:
   ```bash
   python train_cnn.py
   ```
4. Install app dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Launch Streamlit:
   ```bash
   streamlit run app.py
   ```
